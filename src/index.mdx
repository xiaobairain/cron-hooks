## API

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|type|默认定义的表达式类型|Array|['second', 'minute', 'hour', 'day', 'month', 'week']|
|value|给定的表达式|String|0 0 0 \* \* ?|
|style|外部包裹的样式|Object?|{}|
|onChange|监听事件， 可用form托管|Function?|() => {}|
